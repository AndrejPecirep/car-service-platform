# API Overview

## Authentication
- `POST /api/auth/register/`
- `POST /api/auth/login/`

## Resources
- `GET /api/vehicles/`
- `GET /api/services/`
- `GET /api/appointments/`
- `POST /api/appointments/`
- `GET /api/customers/`
- `GET /api/staff/`
- `GET /api/dashboard/summary/`

All resource endpoints require a Bearer token.
